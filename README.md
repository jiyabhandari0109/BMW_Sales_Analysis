# BMW Sales Analysis: Impact of COVID-19

This project explores how COVID-19 affected BMW’s global sales performance using **SQL for exploratory data analysis (EDA)** and **Tableau for interactive visualization**. The dashboard highlights key trends in sales volume, regional performance, model recovery, and fuel-type preferences before, during, and after the pandemic.

---

## Project Overview

The COVID-19 pandemic caused significant disruptions across the automotive industry.  
This analysis focuses on understanding:
- How BMW sales were impacted during COVID years (2020–2021)  
- Which models and regions were most affected  
- How sales recovered post-pandemic  
- Changes in fuel-type preferences over time

---

## Data Insights
Key analyses performed:
- Year-wise and region-wise sales trends  
- Top-selling models per region and year  
- Sales performance before vs. after COVID  
- Recovery rate of models post-2021  
- Fuel-type sales comparison and percentage change
